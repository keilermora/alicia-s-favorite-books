## Minimum requirements

- [Node.js LTS](https://nodejs.org/es/download/)
- [Yarn](https://classic.yarnpkg.com/en/docs/install)

## Init local environment

Run in the project root directory:

```sh
$ yarn install
$ yarn start
```

Then, navigate to `http://localhost:3000/`.
